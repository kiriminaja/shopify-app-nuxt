---
title: AppNav
description: App sidebar navigation. Wraps the Polaris <s-app-nav> web component.
---

## Usage

```vue
<template>
  <ShAppNav>
    <a href="/" rel="home">Home</a>
    <a href="/products">Products</a>
    <a href="/settings">Settings</a>
  </ShAppNav>
</template>
```

::tip
Most apps should use `<ShApp>` instead, which renders `<ShAppNav>` automatically from the `navLinks` config.
::

## Props

This component has no explicit props. All attributes are forwarded to the underlying `<s-app-nav>` element.

## Slots

| Slot      | Description            |
| --------- | ---------------------- |
| `default` | Navigation link items  |

## Polaris Element

Renders as `<s-app-nav>`. See [Polaris AppNav docs](https://shopify.dev/docs/api/app-home/web-components/navigation/app-nav).
