---
title: Option
description: Menu option item. Wraps the Polaris <s-option> web component.
---

## Usage

```vue
<template>
  <ShMenu accessibility-label="Actions">
    <ShOption value="edit" selected>Edit</ShOption>
    <ShOption value="delete" disabled>Delete</ShOption>
  </ShMenu>
</template>
```

## Props

| Prop              | Type      | Default | Description                     |
| ----------------- | --------- | ------- | ------------------------------- |
| `value`           | `string`  | —       | Option value                    |
| `selected`        | `boolean` | —       | Whether the option is selected  |
| `defaultSelected` | `boolean` | —       | Default selected (uncontrolled) |
| `disabled`        | `boolean` | —       | Disables the option             |

## Slots

| Slot      | Description   |
| --------- | ------------- |
| `default` | Option label  |

## Polaris Element

Renders as `<s-option>`. See [Polaris Option docs](https://shopify.dev/docs/api/app-home/web-components/navigation/option).
