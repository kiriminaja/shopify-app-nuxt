---
title: Menu
description: Dropdown menu. Wraps the Polaris <s-menu> web component.
---

## Usage

```vue
<template>
  <ShMenu accessibility-label="Actions">
    <ShOption value="edit">Edit</ShOption>
    <ShOption value="delete">Delete</ShOption>
  </ShMenu>
</template>
```

## Props

| Prop                 | Type     | Default | Description      |
| -------------------- | -------- | ------- | ---------------- |
| `accessibilityLabel` | `string` | —       | Accessible label |

## Slots

| Slot      | Description                   |
| --------- | ----------------------------- |
| `default` | `<ShOption>` menu items       |

## Polaris Element

Renders as `<s-menu>`. See [Polaris Menu docs](https://shopify.dev/docs/api/app-home/web-components/navigation/menu).
