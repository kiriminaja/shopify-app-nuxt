---
title: Divider
description: Visual divider between content. Wraps the Polaris <s-divider> web component.
---

## Usage

```vue
<template>
  <ShStack>
    <p>Above</p>
    <ShDivider />
    <p>Below</p>
  </ShStack>
</template>
```

## Props

| Prop        | Type                     | Default | Description       |
| ----------- | ------------------------ | ------- | ----------------- |
| `direction` | `'inline' \| 'block'`   | —       | Divider direction |
| `color`     | `'base' \| 'strong'`    | —       | Divider color     |

## Slots

| Slot      | Description      |
| --------- | ---------------- |
| `default` | Optional content |

## Polaris Element

Renders as `<s-divider>`. See [Polaris Divider docs](https://shopify.dev/docs/api/app-home/web-components/layout/divider).
