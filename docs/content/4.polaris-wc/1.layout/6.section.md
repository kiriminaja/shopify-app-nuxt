---
title: Section
description: Section container with optional heading. Wraps the Polaris <s-section> web component.
---

## Usage

```vue
<template>
  <ShSection heading="Details" padding="base">
    <p>Section content</p>
  </ShSection>
</template>
```

## Props

| Prop                 | Type               | Default | Description         |
| -------------------- | ------------------ | ------- | ------------------- |
| `heading`            | `string`           | ‚Äî       | Section heading     |
| `padding`            | `'base' \| 'none'` | ‚Äî       | Section padding     |
| `accessibilityLabel` | `string`           | ‚Äî       | Accessible label    |

## Slots

| Slot      | Description        |
| --------- | ------------------ |
| `default` | Section content    |

## Polaris Element

Renders as `<s-section>`. See [Polaris Section docs](https://shopify.dev/docs/api/app-home/web-components/layout/section).
