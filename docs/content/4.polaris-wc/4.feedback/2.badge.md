---
title: Badge
description: Status badge. Wraps the Polaris <s-badge> web component.
---

## Usage

```vue
<template>
  <ShBadge content="Active" tone="success" />
  <ShBadge content="3" size="large" />
</template>
```

## Props

| Prop       | Type                | Default | Description       |
| ---------- | ------------------- | ------- | ----------------- |
| `content`  | `string`            | —       | Badge text        |
| `tone`     | `string`            | —       | Color tone        |
| `progress` | `string`            | —       | Progress indicator|
| `icon`     | `PolarisIcon`       | —       | Badge icon        |
| `size`     | `'base' \| 'large'` | —       | Badge size        |

## Slots

| Slot      | Description           |
| --------- | --------------------- |
| `default` | Additional content    |

## Polaris Element

Renders as `<s-badge>`. See [Polaris Badge docs](https://shopify.dev/docs/api/app-home/web-components/feedback/badge).
