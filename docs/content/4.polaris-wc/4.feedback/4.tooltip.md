---
title: Tooltip
description: Tooltip overlay. Wraps the Polaris <s-tooltip> web component.
---

## Usage

```vue
<template>
  <ShTooltip id="my-tooltip">
    Helpful description text
  </ShTooltip>
</template>
```

## Props

| Prop | Type     | Required | Default | Description         |
| ---- | -------- | -------- | ------- | ------------------- |
| `id` | `string` | Yes      | —       | Tooltip identifier  |

## Slots

| Slot      | Description      |
| --------- | ---------------- |
| `default` | Tooltip content  |

## Polaris Element

Renders as `<s-tooltip>`. See [Polaris Tooltip docs](https://shopify.dev/docs/api/app-home/web-components/feedback/tooltip).
