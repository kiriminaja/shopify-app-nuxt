---
title: Spinner
description: Loading spinner. Wraps the Polaris <s-spinner> web component.
---

## Usage

```vue
<template>
  <ShSpinner accessibility-label="Loading" size="large" />
</template>
```

## Props

| Prop                 | Type                                  | Default | Description      |
| -------------------- | ------------------------------------- | ------- | ---------------- |
| `accessibilityLabel` | `string`                              | —       | Accessible label |
| `size`               | `'base' \| 'large' \| 'large-100'`   | —       | Spinner size     |

## Slots

| Slot      | Description           |
| --------- | --------------------- |
| `default` | Additional content    |

## Polaris Element

Renders as `<s-spinner>`. See [Polaris Spinner docs](https://shopify.dev/docs/api/app-home/web-components/feedback/spinner).
