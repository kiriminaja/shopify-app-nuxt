---
title: QueryContainer
description: Query container for responsive layouts. Wraps the Polaris <s-query-container> web component.
---

## Usage

```vue
<template>
  <ShQueryContainer container-name="sidebar">
    <p>Content that responds to container queries</p>
  </ShQueryContainer>
</template>
```

## Props

| Prop            | Type     | Default | Description       |
| --------------- | -------- | ------- | ----------------- |
| `containerName` | `string` | —       | Container name    |

## Slots

| Slot      | Description        |
| --------- | ------------------ |
| `default` | Container content  |

## Polaris Element

Renders as `<s-query-container>`.
