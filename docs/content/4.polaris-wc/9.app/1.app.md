## Usage

Wrap your app pages with `<ShApp>` to automatically render the [App Bridge navigation menu](https://shopify.dev/docs/api/app-home/app-bridge-web-components/app-nav) from your module config:

```vue [layouts/default.vue]
<template>
  <ShApp>
    <slot />
  </ShApp>
</template>
```

Override the links per-instance:

```vue
<ShApp :links="[{ label: 'Home', href: '/', rel: 'home' }]">
  <slot />
</ShApp>
```

## Props

| Prop    | Type         | Default                                 | Description                     |
| ------- | ------------ | --------------------------------------- | ------------------------------- |
| `links` | `NavLink[]`  | From `runtimeConfig.public.shopify.navLinks` | Navigation links for the sidebar |

```ts
interface NavLink {
  label: string
  href: string
  rel?: 'home' | string
}
```

## Slots

| Slot      | Description               |
| --------- | ------------------------- |
| `default` | App page content          |

## Notes

- Renders a `<ShAppNav>` and `<ShLink>` elements automatically based on the `links` prop or the `navLinks` module config.
- This is a composite component — it does not map to a single Polaris web component.
