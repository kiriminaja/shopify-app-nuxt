---
title: TableRow
description: Table row. Wraps the Polaris <s-table-row> web component.
---

## Usage

```vue
<template>
  <ShTableBody>
    <ShTableRow click-delegate="row-link">
      <ShTableCell>Product A</ShTableCell>
      <ShTableCell>$10.00</ShTableCell>
    </ShTableRow>
  </ShTableBody>
</template>
```

## Props

| Prop            | Type     | Default | Description              |
| --------------- | -------- | ------- | ------------------------ |
| `clickDelegate` | `string` | —       | Delegate click to target |

## Slots

| Slot      | Description    |
| --------- | -------------- |
| `default` | Table cells    |

## Polaris Element

Renders as `<s-table-row>`.
