---
title: TableHeaderRow
description: Table header row. Wraps the Polaris <s-table-header-row> web component.
---

## Usage

```vue
<template>
  <ShTableHeader>
    <ShTableHeaderRow>
      <ShTableCell>Column 1</ShTableCell>
      <ShTableCell>Column 2</ShTableCell>
    </ShTableHeaderRow>
  </ShTableHeader>
</template>
```

## Props

This component has no explicit props. All attributes are forwarded to the underlying element.

## Slots

| Slot      | Description         |
| --------- | ------------------- |
| `default` | Header cell content |

## Polaris Element

Renders as `<s-table-header-row>`.
