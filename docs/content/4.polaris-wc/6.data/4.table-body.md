---

## Usage

```vue
<template>
  <ShTable>
    <ShTableBody>
      <ShTableRow>
        <ShTableCell>Row 1</ShTableCell>
      </ShTableRow>
    </ShTableBody>
  </ShTable>
</template>
```

## Props

This component has no explicit props. All attributes are forwarded to the underlying element.

## Slots

| Slot      | Description    |
| --------- | -------------- |
| `default` | Table rows     |

## Polaris Element

Renders as `<s-table-body>`.
