---
title: Image
description: Optimized image display. Wraps the Polaris <s-image> web component.
---

## Usage

```vue
<template>
  <ShImage original-src="https://example.com/photo.jpg" alt-text="Product photo" />
</template>
```

## Props

| Prop          | Type     | Default | Description          |
| ------------- | -------- | ------- | -------------------- |
| `id`          | `string` | —       | Element ID           |
| `originalSrc` | `string` | —       | Image source URL     |
| `altText`     | `string` | —       | Alternative text     |

## Slots

| Slot      | Description           |
| --------- | --------------------- |
| `default` | Additional content    |

## Polaris Element

Renders as `<s-image>`. See [Polaris Image docs](https://shopify.dev/docs/api/app-home/web-components/content/image).
