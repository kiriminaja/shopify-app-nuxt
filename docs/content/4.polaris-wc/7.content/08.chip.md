---
title: Chip
description: Chip/tag element. Wraps the Polaris <s-chip> web component.
---

## Usage

```vue
<template>
  <ShChip color="blue">Tag 1</ShChip>
</template>
```

## Props

| Prop                 | Type     | Default | Description      |
| -------------------- | -------- | ------- | ---------------- |
| `color`              | `string` | —       | Chip color       |
| `accessibilityLabel` | `string` | —       | Accessible label |

## Slots

| Slot      | Description   |
| --------- | ------------- |
| `default` | Chip content  |

## Polaris Element

Renders as `<s-chip>`. See [Polaris Chip docs](https://shopify.dev/docs/api/app-home/web-components/content/chip).
