---
title: ButtonGroup
description: Group of related buttons. Wraps the Polaris <s-button-group> web component.
---

## Usage

```vue
<template>
  <ShButtonGroup gap="base">
    <ShButton variant="primary">Save</ShButton>
    <ShButton>Cancel</ShButton>
  </ShButtonGroup>
</template>
```

## Props

| Prop                 | Type               | Default | Description         |
| -------------------- | ------------------ | ------- | ------------------- |
| `gap`                | `'base' \| 'none'` | —       | Gap between buttons |
| `accessibilityLabel` | `string`           | —       | Accessible label    |

## Slots

| Slot      | Description       |
| --------- | ----------------- |
| `default` | Button children   |

## Polaris Element

Renders as `<s-button-group>`. See [Polaris ButtonGroup docs](https://shopify.dev/docs/api/app-home/web-components/actions/button-group).
