---
title: ColorPicker
description: Color picker with v-model support. Wraps the Polaris <s-color-picker> web component.
---

## Usage

```vue
<script setup>
const color = ref('')
</script>

<template>
  <ShColorPicker v-model="color" alpha />
</template>
```

## Props

| Prop           | Type      | Default | Description                      |
| -------------- | --------- | ------- | -------------------------------- |
| `modelValue`   | `string`  | —       | Bound value (use with `v-model`) |
| `alpha`        | `boolean` | —       | Enable alpha channel             |
| `defaultValue` | `string`  | —       | Default value (uncontrolled)     |
| `name`         | `string`  | —       | Form field name                  |

## Events

| Event               | Payload  | Description                            |
| ------------------- | -------- | -------------------------------------- |
| `update:modelValue` | `string` | Emitted when value changes (v-model)   |
| `change`            | `Event`  | Native change event                    |
| `input`             | `Event`  | Native input event                     |

## Slots

| Slot      | Description           |
| --------- | --------------------- |
| `default` | Additional content    |

## Polaris Element

Renders as `<s-color-picker>`. See [Polaris ColorPicker docs](https://shopify.dev/docs/api/app-home/web-components/forms/color-picker).
