## App Bridge UI Components (`ShUi*`)

These wrap `ui-*` web components that render **outside** your app's iframe in the Shopify Admin shell. They are the **Vue replacements** for [`@shopify/app-bridge-react`](https://www.npmjs.com/package/@shopify/app-bridge-react) components (`Modal`, `TitleBar`, `SaveBar`, `NavMenu`).

| Component        | App Bridge Element | Description                              |
| ---------------- | ------------------ | ---------------------------------------- |
| `<ShUiModal>`    | `<ui-modal>`       | Full-screen modal outside the iframe     |
| `<ShUiTitleBar>` | `<ui-title-bar>`   | Admin page title bar with action buttons |
| `<ShUiSaveBar>`  | `<ui-save-bar>`    | Contextual save/discard bar              |
| `<ShUiNavMenu>`  | `<ui-nav-menu>`    | Admin sidebar navigation menu            |

### `<ShUiModal>` Props & Events

| Prop      | Type                                    | Default  | Description                                            |
| --------- | --------------------------------------- | -------- | ------------------------------------------------------ |
| `id`      | `string`                                | —        | Unique identifier (used with `shopify.modal.show(id)`) |
| `variant` | `'small' \| 'base' \| 'large' \| 'max'` | `'base'` | Modal size                                             |
| `src`     | `string`                                | —        | URL to load in an iframe inside the modal              |

| Event  | Description                   |
| ------ | ----------------------------- |
| `show` | Emitted when the modal opens  |
| `hide` | Emitted when the modal closes |

Control via App Bridge: `shopify.modal.show(id)`, `shopify.modal.hide(id)`, `shopify.modal.toggle(id)`

### `<ShUiTitleBar>` Props

| Prop    | Type     | Description                                |
| ------- | -------- | ------------------------------------------ |
| `title` | `string` | The title displayed in the Admin title bar |

### `<ShUiSaveBar>` Props & Events

| Prop                  | Type      | Description                           |
| --------------------- | --------- | ------------------------------------- |
| `id`                  | `string`  | Unique identifier                     |
| `discardConfirmation` | `boolean` | Show a confirmation dialog on discard |

| Event  | Description                            |
| ------ | -------------------------------------- |
| `show` | Emitted when the save bar appears      |
| `hide` | Emitted when the save bar is dismissed |

Control via App Bridge: `shopify.saveBar.show(id)`, `shopify.saveBar.hide(id)`, `shopify.saveBar.toggle(id)`, `shopify.saveBar.leaveConfirmation()`
